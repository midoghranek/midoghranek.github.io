HTML Generator

Open this file Generators > html_generator > index.html in your browser.

Custom Style Settings

  1. Here you can enter your google font name, if you haven't one click on Get one link. note if the font name contain two words ex: Open Sans, You must write it with the space in between ex: Open Sans, Droid Sans etc..
  2. Choose between LTR or RTL.
  3. Input in numbers only icon height ex: 180, 190, 200.
  4. Click inside checkbox if you install fontawesome lib before in your site.
  5. Choose between 1, 2, 3 or 4 rows of inline icons.
  6. Input font size in number for icon
  7. Input font size in number for icon in hover
  8. Input font size in number for number
  9. Input font size in number for text after number
  10. Input color Hex code for icon hover color
  11. Click here to fill boxs with default values
  12. Click on Edit button after save button to re-enter new values
  13. Click on save button to add edited or default values to the result code

Icon settings

  1. Choose between 3 styles of your icon
  2. Enter your fontawesome icon name, you can write it with fa- or not the form auto detect it and complete it automatically, click on Get One link to log to fontawesome.io site.
  3. Here you can enter icon color hex code, click on Get one to get offical codes of the most popular brands
  4. Enter your followers number for this site
  5. Enter your followers text apear under number text ex: followers, fans, likes
  6. Your social icon onclick URL
  7. Click on Add icon button to add the icon to your collection

CSS code

Install css files to your site.

<link rel="stylesheet" href="css/sicons-style.css" />
<link rel="stylesheet" href="css/font-awesome/css/font-awesome.min.css" />

Customize CSS

/*********************************
====>  Start CSS customization 
**********************************/

    /* Font */
 
    @import url(http://fonts.googleapis.com/css?family=Droid+Serif:400,700);

    #social {
        font-family: 'Droid Serif', serif;
    }

    /* RTL */ 

    #social li {
        float: left;
    }


    /* Icon height */ 

  #social li a {
    height: 150px;
  }
  
    /* Icons row */ 

  #social li {
    width: 33.333%;
  }

    /* font size */ 

  #social li a .fa {
    font-size: 40px;
  }
  
  #social li a:hover .fa {
    font-size: 50px;
  }
  
  #social li a .item-count {
    font-size: 21px;
  }
  
  #social li a .item-text {
    font-size: 11px;
  }

    /* Icons hover color */ 

  #social li a:hover {
    background-color: #333!important;
  }

/*********************************
End CSS customization <====
**********************************/

Open css > sicons-style.css in your text editor.

  1. Droid+Serif: replace with google font name, if the name contain two words remove space with plus sign.
  2. Droid Serif: replace with google font name, if the name contain two words don't reomve the space.
  3. left: replace left with right for RTL support.
  4. 150: replace 150 with your icon height.
  5. 33.333: replace it with your % with to the icon inline, use 25 for 4 icons, 33.333 for 3 icons, 50 for 2 icons or 100 for 1 icon.
  6. 40: change with font size of icon
  7. 50: change with font size of icon on hover
  8. 21: change with font size of icon number of followers text.
  9. 11: change with font size of icon text after number of followes ex: like, followers.
